Tetris

This tutorial will explain what this app is and walk you through the features

You can jump in immediately by pressing "skip tutorial", or click "next" to access the next portion of the tutorial

Controls

PC:

Hold Block: Press the 'h' key

Rotate Block: Press the up arrow key

Shift Block: Press the corresponding arrow key

Drop Block: Press the spacebar

Mobile:

Use the buttons at the bottom of the screen

Changing the Speed

The value under the "Drop Delay" menu item represents the amount of time the game will wait between letting the block fall another bit

You can use the slider to adjust this waiting time in milliseconds

A smaller value will mean that there's less time between the block falling, making it fall faster

AI

If enabled, the AI will automatically decide where to put the block and how to rotate it. This can be easily overriden using the game controls

You can toggle the AI using the toggle switch in the top left corner
